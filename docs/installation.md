# Installation

ELRS Mobile is available for both Android and iOS via their respective beta testing programs.

## Android (Google Play)
The highly recommended way to install on Android is via the **Google Play Store (Open Testing)**. This ensures you receive automatic updates.
- **Join:** [ELRS Mobile on Google Play](https://play.google.com/store/apps/details?id=io.datarx.elrsmobile)

## iOS (Apple TestFlight)
The app is currently in Beta for iOS. Join the TestFlight program to install it on your iPhone or iPad.
- **Join:** [ELRS Mobile on TestFlight](https://testflight.apple.com/join/jwU1WrpX)

## Manual APK (Android Only)
If you prefer not to use the Play Store, you can install the latest release directly.
- **Download:** [GitHub Releases](https://github.com/wbhinton/ELRS-Mobile/releases)
- **Install:** Download and install the `.apk` file. *(Note: You may need to enable "Install from Unknown Sources" in your Android settings).*
